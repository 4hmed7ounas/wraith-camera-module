================================================================================
QUICK INSTALL - Fastest Way to Get Running
================================================================================

Problem: Your internet connection is unstable, pip keeps timing out
Solution: Minimal installation of only essential packages

Time needed: 5-10 minutes (instead of 30+ with retries)


================================================================================
STEP 1: ACTIVATE VIRTUAL ENVIRONMENT
================================================================================

.\venv\Scripts\activate

You should see (venv) at the start of your terminal


================================================================================
STEP 2: INSTALL MINIMAL PACKAGES
================================================================================

Copy-paste this single command:

pip install --retries 5 --default-timeout 500 numpy opencv-python ultralytics


That's it. Just 3 packages, no paddleocr (which is causing the download issues).

Expected output:
  Successfully installed numpy
  Successfully installed opencv-python
  Successfully installed ultralytics
  (+ some dependencies)


================================================================================
STEP 3: RUN LIGHTWEIGHT PLATE READER
================================================================================

python plate_reader_lightweight.py

This version:
‚úì Detects plates in real-time
‚úì Shows bounding boxes
‚úì Displays FPS counter
‚úì NO OCR (no need for paddleocr/paddlepaddle)
‚úì Works with just numpy + opencv + ultralytics


================================================================================
WHAT IF INSTALLATION STILL FAILS?
================================================================================

Try one package at a time:

pip install --retries 5 --default-timeout 500 numpy
pip install --retries 5 --default-timeout 500 opencv-python
pip install --retries 5 --default-timeout 500 ultralytics

If one fails, wait 5 minutes and try again.

The --retries 5 flag will auto-retry if connection drops.


================================================================================
IF opencv-python FAILS (Large file)
================================================================================

Try the headless version:

pip install --retries 5 --default-timeout 500 opencv-python-headless

Same functionality, slightly smaller file size.


================================================================================

python -c "import cv2; print('‚úì OpenCV OK')"

================================================================================
TROUBLESHOOTING
================================================================================

Q: "Module not found: cv2"
A: Run: pip install --retries 5 --default-timeout 500 opencv-python

Q: "Module not found: ultralytics"
A: Run: pip install --retries 5 --default-timeout 500 ultralytics

Q: "Connection timed out"
A: Wait 5 minutes, run again. The --retries flag should auto-retry.

Q: "No camera detected"
A:
  - Check camera is plugged in
  - Try: python -c "import cv2; cap=cv2.VideoCapture(0); print(cap.isOpened())"
  - If False, camera not detected

Q: "Model download is slow"
A: This is normal - first run downloads YOLOv8 model (~6MB)
   - Only happens once
   - Subsequent runs are instant
